Basic toolbar button:

```jsx
const Button = require('./Button').default
;<Button>Push Me</Button>
```

Big pink toolbar button:

```jsx
const Button = require('./Button').default
;<Button size="large" color="deeppink">
  Lick Me
</Button>
```

Disabled toolbar button:

```jsx
const Button = require('./Button').default
;<Button size="large" disabled>
  You can't click me
</Button>
```
